Logical Operators

Logical operators in Python:

  • and: Result is true only when both LHS and RHS are True
  • or: Result is false only when both LHS and RHS are False
  • not: Negates the result

Examples:

    True and True = True
    True or False = True
    not True = False

Assignment Operators

Assignment operators in Python:

  • = (Assignment): Assigns the value of the RHS variable/constant to the LHS variable
  • += (Addition assignment): Adds the RHS to the LHS variable
  • -= (Subtraction assignment): Subtracts the RHS from the LHS variable
  • *= (Multiplication assignment): Multiplies the LHS by the RHS
  • /= (Division assignment): Divides the LHS by the RHS
  • %= (Modulus assignment): Calculates the modulus of LHS and RHS
  • **= (Exponentiation assignment): Calculates LHS to the power of RHS
  • //= (Floor Division assignment): Performs floor division on LHS and RHS

Examples:

    a = 89
    b = 7
    c = 8
    a += 4 # Equivalent to a = a + 4
    b -= 1 # Equivalent to b = b - 1
    c *= 3 # Equivalent to c = c * 3

Floor, Ceil, Round

These are functions used to manipulate numeric values:

  • math.floor(x): Returns the largest integer less than or equal to x.
  • math.ceil(x): Returns the smallest integer greater than or equal to x.
  • round(x): Rounds x to the nearest integer.

Examples:

    math.floor(3.6) = 3
    math.ceil(3.6) = 4
    round(3.6) = 4

Logical Operators

Logical operators in Python:

Membership Operators

Membership operators in Python:

  • in: Returns True if a value is found in the sequence.
  • not in: Returns True if a value is not found in the sequence.

Examples:

    3 in [1, 2, 3] = True
    'apple' not in ['banana', 'cherry'] = True

Strings

A string is a group of characters (alphabets, numerics, special characters, spaces) and is represented using single quotes ('') and double quotes ("").

If you start with a single quote, you must end with a single quote, and if you start with a double quote, you must end with a double quote.

The usual convention is to use single quotes for words and double quotes for sentences. The outermost quotes specify the start and end of the string, and they are not counted when calculating the length.

Examples:

    'Python is easy to learn'
    "Python is easy to learn"
    "Python's syntax is simple"
    'She said, "Hello!"'

String Indexing

Strings in Python are indexed, and indexing starts from 0. You can access individual characters using square brackets.

The syntax for indexing is variable_name[index].

Examples:

    text = "Python"
    text[0] = 'P'
    text[2] = 't'

String Slicing

String slicing allows you to extract substrings from strings using the syntax string_variable_name[start_index : end_index].

The result includes the character at the start index and excludes the character at the end index.

Examples:

    text = "Python is easy"
    text[0:6] = 'Python'
    text[7:9] = 'is'

Type Casting

Type casting is the process of converting one data type into another. In Python, you can use built-in functions to perform type casting:

  • int(x): Converts x to an integer.
  • float(x): Converts x to a floating-point number.
  • str(x): Converts x to a string.
  • bool(x): Converts x to a boolean value.

Examples:

    int("42") = 42
    float(3) = 3.0
    str(123) = '123'
    bool(0) = False

Data Types

Data types in Python specify what kind of values a variable can hold. Here are some common data types:

  • int: Represents integers (whole numbers).
  • float: Represents floating-point numbers (decimal numbers).
  • str: Represents strings (sequences of characters).
  • bool: Represents Boolean values (True or False).

Arithmetic Operators

Arithmetic operators perform mathematical operations on numeric values. Common arithmetic operators include:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus, returns the remainder)
  • ** (Exponentiation)
  • // (Floor Division, rounds down to the nearest integer)

Examples:

    x = 10
    y = 3
    addition = x + y      # 13
    subtraction = x - y   # 7
    multiplication = x * y # 30
    division = x / y       # 3.333...
    remainder = x % y      # 1
    exponentiation = x ** y # 1000
    floor_division = x // y # 3

Comparison Operators

Comparison operators are used to compare two values. They return either True or False. Common comparison operators include:

  • == (Equal to)
  • != (Not equal to)
  • < (Less than)
  • > (Greater than)
  • <= (Less than or equal to)
  • >= (Greater than or equal to)

Examples:

    a = 5
    b = 10
    equal = (a == b)         # False
    not_equal = (a != b)     # True
    less_than = (a < b)      # True
    greater_than = (a > b)   # False
    less_equal = (a <= b)    # True
    greater_equal = (a >= b) # False

Math Functions: floor, ceil, round

Python provides several math functions for numeric operations:

  • math.floor(x): Returns the largest integer less than or equal to x.
  • math.ceil(x): Returns the smallest integer greater than or equal to x.
  • round(x): Rounds x to the nearest integer.

Example:

    import math
    value = 3.6
    floor_value = math.floor(value) # 3
    ceil_value = math.ceil(value)   # 4
    rounded_value = round(value)   # 4

Logical Operators

Logical operators are used to combine and manipulate Boolean values. Common logical operators include:

  • and: Returns True if both operands are True.
  • or: Returns True if at least one operand is True.
  • not: Returns the opposite of the operand's value (True becomes False and vice versa).

Examples:

    x = True
    y = False
    result_and = x and y  # False
    result_or = x or y    # True
    result_not = not x    # False

Conditional Logic

Conditional statements allow you to execute different code blocks based on certain conditions. In Python, the primary conditional statement is the "if" statement.

Example:

    age = 20
    if age >= 18:
        print("You are an adult.")
    else:
        print("You are a minor.")

String Basics

Strings are sequences of characters, and they can be represented using single quotes ('') or double quotes ("").

Example:

    name = 'Alice'
    message = "Hello, World!"

String Functions

Python provides many built-in functions for working with strings. Common string functions include:

  • len(string): Returns the length of the string.
  • string.upper(): Converts the string to uppercase.
  • string.lower(): Converts the string to lowercase.
  • string.strip(): Removes leading and trailing whitespace.

Examples:

    text = "Python is fun"
    length = len(text)        # 13
    uppercase_text = text.upper()   # "PYTHON IS FUN"
    lowercase_text = text.lower()   # "python is fun"
    stripped_text = text.strip()     # "Python is fun" (no leading/trailing whitespace)

String Indexing

Strings in Python are indexed, and you can access individual characters by their index (position) within the string. Indexing starts at 0.

Example:

    text = "Python"
    first_char = text[0]   # "P"
    second_char = text[1]  # "y"

String Slicing

String slicing allows you to extract a portion of a string. The syntax is string[start:end]. It includes the character at the start index but excludes the character at the end index.

Example:

    text = "Python is easy"
    substring = text[7:9]   # "is"

    • Nearest integer

    • 3.2 -> 3
  2. Ceil

    • Nearest integer smaller than the given floating value
    • 3.7 -> 3
    • 3.2 -> 3

Importing libraries

Whenever a functionality is not available in Python by default, we will import external libraries.

In the example below, we are importing the math library because ceil and floor are not available in Python by default.

Import will include all the functions available in the external library to the current executing Jupyter notebook.

One of the simple syntaxes to import a library is:

import library_name
    # Example: import math

Only after importing a library can we use its functions.

Use functions with a library by using the library name followed by a dot (.), then the function name.

For one Jupyter notebook execution, importing a library once is sufficient.

import math

    print(round(3.2))
    print(math.ceil(3.2))
    print(math.floor(3.7))

Output:

    3
    4
    3
    
print(math.floor(123.784))

    123
