Python Strings

Strings are sequences of characters, and in Python, they are represented with single quotes (') or double quotes (").

Indexing and Slicing:

Characters in a string can be accessed using indexing. Python uses zero-based indexing, meaning the first character is at index 0. Negative indexing starts from the end of the string (-1 for the last character).

text = "Python" print(text[0]) # Output: P print(text[-1]) # Output: n

Slicing allows you to extract a portion of the string. Syntax: text[start:stop:step]. If start or stop is not specified, it defaults to the beginning/end of the string. step specifies the interval between characters.

text = "Python Programming" print(text[7:18]) # Output: Programming print(text[::-1]) # Output: gnimmargorP (reverses the string)

String Functions:

Python provides various string functions for manipulation:

text = "python is easy" print(text.upper()) # Output: PYTHON IS EASY print(text.find("easy")) # Output: 10 (index where "easy" starts) print(text.replace("easy", "simple")) # Output: python is simple

Escape Characters:

Escape characters are used to represent special characters in strings. For example, '\\n' represents a newline character and '\\t' represents a tab character.

print('Hello\\nWorld') # Output: Hello\nWorld

Raw Strings:

Raw strings, denoted with 'r' before the string, treat backslashes as literal characters, useful for file paths.

path = r'C:\\Users\\username\\Documents\\file.txt' print(path) # Output: C:\\Users\\username\\Documents\\file.txt

Conditional Statements

Conditional statements allow you to make decisions in your code. The if statement checks a condition, and if it's true, executes a block of code.

a = 5 if a > 0: print("a is positive") # Output: a is positive

Loops

For Loop is used for iterating over a sequence (list, tuple, string, etc.).

numbers = [1, 2, 3, 4, 5] for num in numbers: print(num) # Output: 1 2 3 4 5

While Loop is used for repeated execution as long as an expression is true.

count = 0 while count < 5: print(count) count += 1 # Output: 0 1 2 3 4

Typecasting

Typecasting is the process of converting one data type into another.

num_str = "42" num = int(num_str) # Converts string to integer print(num) # Output: 42

Using split() with Paths

    path3 = r'C:\\Users\\adarsha\\OneDrive\\0_MIT_Sept2023_Python\\Day2_23_09_2023.ipynb'
    print(path3.split(r'\\'))
    # Output: ['C:', 'Users', 'adarsha', 'OneDrive', '0_MIT_Sept2023_Python', 'Day2_23_09_2023.ipynb']

    path3 = 'C:/Users/adarsha/OneDrive/0_MIT_Sept2023_Python/Day2_23_09_2023.ipynb'
    print(path3.split(r'\\'))
    # Output: ['C:/Users/adarsha/OneDrive/0_MIT_Sept2023_Python/Day2_23_09_2023.ipynb']

Note: Always use r while working with paths or use os.path.

Type Casting / Type Conversion in Python

Converting String to Integer

    # Converting string to int
    st = '34'
    print(st)
    print(type(st))
    a = int(st)  # Type casting is happening here
    print(a)
    print(type(a))
    # Output:
    # 34
    # <class 'str'>
    # 34
    # <class 'int'>

Converting String with Float to Int (Error Example)

    # Converting string having float to int --- Result in error
    st = '34.5'
    print(st)
    print(type(st))
    try:
        a = int(st)  # Gives error because string does not contain an integer, it contains float
    except ValueError as e:
        print("Error:", e)
    # Output:
    # 34.5
    # <class 'str'>
    # Error: invalid literal for int() with base 10: '34.5'

Converting String with Float to Float

    # Converting string having float to float will work
    st = '34.5'
    print(st)
    print(type(st))
    a = float(st)  # Type casting is happening here
    print(a)
    print(type(a))
    # Output:
    # 34.5
    # <class 'str'>
    # 34.5
    # <class 'float'>

Converting String with Float to Int (Two-Step Conversion)

    # If you want to convert a string having float to int.
    # First convert to float, then convert to int
    st = '34.5'
    print(st)
    print(type(st))
    a = int(float(st))  # Type casting is happening here. First convert to float, then convert to int
    print(a)
    print(type(a))
    # Output:
    # 34.5
    # <class 'str'>
    # 34
    # <class 'int'>

Converting Int or Float to String

    # Convert int or float to string
    a_int = 23
    print(type(a_int))
    print(a_int)

    print(type(a_str))
