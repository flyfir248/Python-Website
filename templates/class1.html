Rules for creating a variable name

  • Start with alphabet or underscore ( _ )
  • Can contain alphabets, underscore, or number. Special characters are not allowed (including space)
  • Case sensitive

Valid variable names

    a = 12
    _a = 123.4
    b34 = 2354

    print(a)
    print(_a)
    print(b34)

Invalid variable names

    1a = 23 # Invalid variable name results in an error
    $a = 23 # Invalid variable name results in an error
    abc! = 23 # Invalid variable name results in an error
    abc% = 23 # Invalid variable name results in an error

Three Different Ways of Converting a Floating Value to an Integer

Round

Nearest integer

    print(round(3.2))

This will output:

    3

Ceil

Nearest integer greater than the given floating value

    print(math.ceil(3.2))

This will output:

    4

Floor

Nearest integer smaller than the given floating value

    print(math.floor(3.7))

This will output:

    3

Importing Libraries

Whenever a functionality is not available in Python by default, you can import external libraries.

In the below example, we are importing 'math' because 'ceil' and 'floor' are not available in Python by default.

Import will include all the functions available in the external library to the current executing Jupyter notebook.

One of the simple syntaxes to import a library is:

    import library_name
    # Example:
    import math

Only after importing a library can you use its functions. Use functions with the library by using the library name followed by a dot (.), then the function name.

For one Jupyter notebook execution, importing a library once is sufficient.

    import math

    print(round(3.2))
    print(math.ceil(3.2))
    print(math.floor(3.7))

This will output:

    3
    4
    3
    
    
    print(math.floor(123.784))

This will output:

    123
