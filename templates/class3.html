Single and Double Quotes

In Python, strings can be defined using single quotes (' ') or double quotes (" ").

For example:

st1 = 'Python is easy to understand'
    st2 = "Python is 'easy' to understand"

You can also use escape characters to include quotes within a string. For example:

st3 = 'In English there is a proverb , "Work is worship".'

To include double quotes within a double-quoted string, use the escape character \".

st4 = "In English there is a proverb , \"Work is worship\"."

To include single quotes within a single-quoted string, use the escape character '.

st4 = 'In English there is a proverb , \'Work is worship\'.'

String Indexing

String indexing in Python starts from 0. You can access individual characters of a string using positive indexing (from the beginning) or negative indexing (from the end).

st = "Double quotes has a special meaning."
    character = st[12]
    print(character)  # Output: 's'

You can also use negative indexing to access characters from the end of the string.

st = "Double quotes has a special meaning."
    character = st[-6]
    print(character)  # Output: 's'

String Slicing

String slicing allows you to extract a portion of a string by specifying a start and end index.

st = "Double quotes has a special meaning."
    substring = st[12:-6]
    print(substring)  # Output: 's has a special meaning.'

String slicing can also be used to reverse a string:

st = "Python"
    reversed_st = st[::-1]
    print(reversed_st)  # Output: 'nohtyP'

String Functions

Python provides various string functions to manipulate strings. For example:

st = "Python is very simple"
    index = st.find('is')  # Find the starting index of 'is' in the string
    print(index)  # Output: 7

    st = "Python is very simple"
    lowercase_st = st.lower()  # Convert the string to lowercase
    print(lowercase_st)  # Output: 'python is very simple'

    st1 = 'Hi. '
    st2 = 'How are you ?'
    combined_st = st1 + st2  # Concatenate two strings
    print(combined_st)  # Output: 'Hi. How are you ?'

Practice Questions

  1. Write a Python program that takes a string as input and uses slicing to reverse the string.
  2. Given a string, create two new strings by slicing the original string. One string should contain characters at even indices, and the other should contain characters at odd indices.
  3. Write a Python function that takes a string as input and extracts alternate characters to create a new string.
  4. Write a Python program that extracts and prints the domain from an email address.
  5. Create a Python program that takes a string as input and swaps the first and last characters.
  6. Create a Python program that takes a string as input and reflects it by repeating it backward.

Python Practice Questions

1. Reverse a String

To reverse a string using slicing:


      input_string = "Hello, World!"
      reversed_string = input_string[::-1]
      print(reversed_string)

Output:


      !dlroW ,olleH

2. Create Two New Strings with Even and Odd Indices

To create two new strings with even and odd indices:


      input_string = "Python"
      even_indices_string = input_string[::2]
      odd_indices_string = input_string[1::2]
      print(even_indices_string)
      print(odd_indices_string)

Output:


      Pto
      yhn

3. Extract Alternate Characters

To extract alternate characters from a string:


      input_string = "Python"
      alternate_characters = input_string[::2]
      print(alternate_characters)

Output:


      Pto

4. Extract Domain from Email Address

To extract the domain from an email address:


      email = "user@example.com"
      domain = email.split('@')[1]
      print(domain)

Output:


      example.com

5. Swap First and Last Characters

To swap the first and last characters of a string:


      input_string = "Python"
      swapped_string = input_string[-1] + input_string[1:-1] + input_string[0]
      print(swapped_string)

Output:


      nythoP

6. Reflect and Repeat a String

To reflect and repeat a string:


      input_string = "Python"
      reflected_string = input_string + input_string[::-1]
      print(reflected_string)

Output:


      PythonnohtyP
