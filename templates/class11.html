What is a function?

  • A function is a reusable code snippet that performs a specific operation.
  • Why do we need functions?
    • Code reuse
    • Better modularity and readability
  • Components of a function:
    • Function definition
      • Function name: Follows variable naming rules
      • Parameters: Input values (optional)
    • Function call
  • How do we write a function?
    • def function_name(parameters-optional):
          body of the function
          return-optional

      return_value = function_name(arguments)

Types of Arguments

Positional Arguments

def f1(a, b):
          print(a)
          print(b)
f1(89, 1)  # Output:
      # 89
      # 1
f1(1)  # Will result in an error because b is mandatory

Default Arguments

When the default value is not set, we must pass the value during the function call. When the default value is set, no need to pass the value during the function call. If you pass a value, it will be assigned based on the position.

def f1(a, b=20):
          print(a)
          print(b)
f1(78)  # If b is not passed, the default value will be assigned
f1(89, 100)  # If b is passed, the argument passed will be assigned
      # Output:
      # 89
      # 100
f1(89, 1009898989)  # Output:
      # 89
      # 1009898989
f1(67)  # Output:
      # 67
      # 20 (default value for b)

          res = 0
          for i in args:
              res = res + i
          return res

          res = 1
