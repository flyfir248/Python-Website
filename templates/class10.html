  • Different ways of importing functions from a module
    • import module_name:
      • Entire library functions and variables are loaded but existing functions will not be overwritten because we need to access them through module_name followed by dot
    • import module_name as alias:
      • Entire library functions and variables are loaded but existing functions will not be overwritten because we need to access them through alias followed by dot
    • from module_name import *:
      • Entire library functions and variables are loaded into the current notebook/module as it is. This may overwrite existing functions/variables. Not recommended.
    • from module_name import function1, function2:
      • Can import only required functions without loading the entire library. This is most commonly used because it avoids overwriting existing functions/variables.
    • from module_name import function1 as alias:
      • Similar to the above import.

    • Filter numbers divisible by 3 from a list of integers

      • ip_li = [1, 7, 5, 4, 9, 22]
  op_li = ['even' if z % 2 == 0 else 'odd' for z in ip_li]

Python List Comprehension Examples

Return even or odd only for numbers greater than 5. Also, return number along with even or odd:

ip_li = [3, 4, 5, 67, 1, 2, 3, 9, 34]
  op_li = [[z, 'even' if z % 2 == 0 else 'odd'] for z in ip_li if z > 5]
  console.log(op_li);  // Output: [[67, 'odd'], [9, 'odd'], [34, 'even']]

For a given list of numbers return their powers of 3:

ip_li = [3, 4, 5]
  op_li = [y ** 3 for y in ip_li]
  console.log(op_li);  // Output: [27, 64, 125]

Remove stop words from a list of words using list comprehension:

ip_string = "The control room at the Bruhat Bengaluru Mahanagara Palike headquarters - the nerve centre of disaster management in the event of rain and flooding - appears to be hit by absenteeism."
  words = ip_string.split()
  stop_words = ['the', 'of', 'at', 'or', 'in', 'and', 'by', 'to']
  punctuations = ['-', '.']
  filtered_words = [w for w in words if w.toLowerCase() not in stop_words.concat(punctuations)]
  console.log(filtered_words);
  // Output: ['control', 'room', 'Bruhat', 'Bengaluru', 'Mahanagara', 'Palike', 'headquarters', 'nerve', 'centre', 'disaster', 'management', 'event', 'rain', 'flooding', 'appears', 'be', 'hit', 'absenteeism.']

Extract words that are not present in other sentences:

list_of_sentences = ["This is a simple example ", "This NLP exercise is fun and simple", "Practice makes perfect"]
  res = []
  for (let i = 0; i < list_of_sentences.length; i++) {
      let other_sent = list_of_sentences.slice(0, i).concat(list_of_sentences.slice(i + 1)).join(' ').split(' ');
      let sentenceWords = list_of_sentences[i].split(' ');
      for (let j = 0; j < sentenceWords.length; j++) {
          if (!other_sent.includes(sentenceWords[j])) {
              res.push(sentenceWords[j]);
          }
      }
  }
  console.log(res);
  // Output: ['a', 'example', 'NLP', 'exercise', 'fun', 'and', 'Practice', 'makes', 'perfect']
