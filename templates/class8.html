What is a Function?

Function is a code snippet which implements a specific operation.

Why Do We Need Functions?

  • Code Reuse
  • Better Modularity and Readability

Components of a Function

  • Function Definition
  • Function Name (can be anything following variable naming rules)
  • Function Call

How Do We Write a Function?

def function_name(parameters-optional):
      # Body of the function
      return-optional

Function Examples

def add(num1, num2):
      return num1 + num2

  def subtract(num1, num2):
      return num1 - num2

  # Function Call
  result = add(5, 3)
  print(result)

  result = subtract(8, 2)
  print(result)

Types of Arguments

  • Positional Arguments (Mandatory)
  • Default Arguments (Optional)
  • Keyword Arguments
  • Variable Length Arguments
  • Variable Length Keyword Arguments

Positional Arguments

def example_function(a, b):
      print(a, b)

  example_function(1, 2)

Default Arguments

def example_function(a, b=10):
      print(a, b)

  example_function(5) # Output: 5 10

Keyword Arguments

def example_function(a, b):
      print(a, b)

  example_function(b=20, a=15) # Output: 15 20

Variable Length Arguments

def example_function(*args):
      print(args)

  example_function(1, 2, 3, 4) # Output: (1, 2, 3, 4)

Variable Length Keyword Arguments

def example_function(**kwargs):
      print(kwargs)

  example_function(a=1, b=2, c=3) # Output: {'a': 1, 'b': 2, 'c': 3}

Sorting with Functions

def sort_function(value):
      return value[1]

  list_of_lists = [[1, 'apple'], [3, 'banana'], [2, 'cherry']]
  list_of_lists.sort(key=sort_function)
  print(list_of_lists)

Function for Case Study

def perform_operation(num1, num2, operation):
      if operation == 'sum':
          return num1 + num2
      elif operation == 'prod':
          return num1 * num2
      elif operation == 'sub':
          return num1 - num2
      elif operation == 'div':
          if num2 == 0:
              return 'Cannot divide a number by 0'
          else:
              return num1 / num2

Python Function Arguments

Types of Arguments

Python functions can have different types of arguments:

1. Positional Arguments (Mandatory)

Positional arguments are mandatory and must be passed in the same order as they are defined in the function.

def example_function(a, b):
      # Function body

2. Default Arguments (Optional)

Default arguments have default values set in the function definition. If a value is not provided during the function call, the default value is used.

def example_function(a, b=10):
      # Function body

3. Keyword Arguments

Keyword arguments are passed with their corresponding parameter names. This way, the order of the arguments does not matter.

example_function(b=20, a=15)

4. Variable Length Arguments

Variable length arguments allow a function to accept any number of positional arguments. The arguments are collected into a tuple.

def example_function(*args):
      # Function body

5. Variable Length Keyword Arguments

Variable length keyword arguments allow a function to accept any number of keyword arguments. The arguments are collected into a dictionary.

def example_function(**kwargs):
      # Function body

Examples:

Example 1: Positional and Default Arguments

def greet(name, greeting="Hello"):
      print(greeting, name)

  greet("Alice")  # Output: Hello Alice
  greet("Bob", "Hi")  # Output: Hi Bob

Example 2: Variable Length Arguments

def calculate_sum(*args):
      total = 0
      for num in args:
          total += num
      return total

  result = calculate_sum(1, 2, 3, 4, 5)
  print(result)  # Output: 15

Example 3: Variable Length Keyword Arguments

def print_details(**kwargs):
      for key, value in kwargs.items():
          print(key + ": " + value)

  print_details(name="Alice", age="30", city="Wonderland")
  # Output:
  # name: Alice
  # age: 30
  # city: Wonderland

Example 4: Sorting a List of Tuples using a Function

def sort_list(val):
      return val[1]

  indian_cricketers = [
      ["Virat Kohli", 33, 12169],
      ["Rohit Sharma", 34, 9115],
      ["Shikhar Dhawan", 35, 5978],
      ["Ajinkya Rahane", 33, 5185],
      ["Cheteshwar Pujara", 34, 6267],
  ]

  indian_cricketers.sort(key=sort_list)
  print(indian_cricketers)
  # Output: Sorted list based on the second element of each tuple (age)
